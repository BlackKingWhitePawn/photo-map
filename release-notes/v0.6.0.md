# Traverse v0.6.0

Date: 2026-07-16

This release improves the map marker experience for large local photo libraries. Markers are now rendered with a Compose overlay above MapLibre so visible clusters and photo thumbnails remain on screen even when style layers do not draw them.

## Added

* Compose overlay rendering for map markers.
* Thumbnail markers for visible clusters and single photos.
* A map debug panel with visible cluster count and cluster coordinates.
* A settings toggle to show or hide the debug panel.
* Unit-test coverage for marker render rules and debug information formatting.

## Fixed

* Clusters and thumbnails no longer disappear because of MapLibre style-layer rendering.
* Marker screen positions update while the camera moves, instead of staying fixed until camera idle.
* A single photo in a stored cluster is rendered as a photo marker, not as a cluster circle.
* Cluster counts are shown as small badges so they do not cover the marker thumbnail.
* Overlay marker taps are handled directly: clusters zoom the map, single photos open the preview panel.

## Known Limitations

* The marker overlay is a pragmatic rendering layer above MapLibre. The MapLibre source/layer path remains in the codebase, but the overlay is the visible marker UI for this release.
* Room, Coil gallery grid, fullscreen viewer, and WorkManager background scan are still outside this release prep.
