# Photo Map v0.8.1

Date: 2026-07-17

## Added

* Date picker opens from the start and end date labels in the range slider.
* Month and year tick marks are drawn behind the date range slider.

## Changed

* The date range slider scales to the selected date span when a filter is active.
* Dragging slider handles toward the visible range edges expands the scale back toward the full date span.
* Slider handles snap to nearby month, year, and range boundary ticks.

## Fixed

* The map no longer shows the "Нет фотографий с геопозицией" empty state when an active date filter hides all photos.
