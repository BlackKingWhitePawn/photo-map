# Ветка `feature-08-map-cluster-tuning`

## Release Target

```text
v0.6.0
```

## Цель

Доработать кластеризацию миниатюр на карте так, чтобы на любом zoom были видны управляемые группы фотографий, при этом фотографии дальше 50 км друг от друга не объединялись в один кластер.

## Границы работ

В ветку входит:

* собственная экранная кластеризация для MapLibre-слоя кластеров;
* автоматическое усиление радиуса кластеризации на низком zoom, включая уровень города;
* жесткое ограничение расстояния внутри кластера: не больше 50 км между любой парой фотографий;
* отдельный некластеризованный source для видимых миниатюр;
* настраиваемые параметры алгоритма с сохранением на устройстве;
* открытие фотографии в системной галерее по нажатию на миниатюру в нижней панели;
* логи по количеству рассчитанных кластеров, одиночных точек, видимых миниатюр, zoom и эффективному радиусу.

В ветку не входит:

* запуск сборки Codex-ом;
* запуск тестов, lint, tsc, eslint или npx;
* изменение или удаление оригинальных фотографий;
* отправка фотографий, координат или EXIF наружу.

## Проверки

Сборку выполняет пользователь вручную. Codex не запускает build/compile/test после патча.
