# Traverse v0.1.0

Дата: 2026-07-15

Первый bootstrap-релиз проекта.

## Добавлено

* Инициализирован Android-проект `Traverse`.
* Добавлена базовая Gradle-конфигурация.
* Добавлен модуль `app` с package name `com.example.photomap`.
* Добавлен минимальный Compose-интерфейс.
* Подготовлены правила веток, коммитов, версионирования и релизов.
* Добавлены `branch.md`, `release.md` и `CHANGELOG.md`.
* Настроен `.gitignore` для Android-сборок, APK/AAB, signing-файлов, IDE-файлов и логов.

## APK

Ожидаемое имя release asset:

```text
photomap-v0.1.0.apk
```

Перед публикацией APK нужно пересобрать signed release APK после изменения `versionName` на `0.1.0`.

## Проверки

Не запускались в рамках подготовки release notes по текущему запросу.

## Известные ограничения

* MediaStore ещё не реализован.
* Runtime-разрешения ещё не реализованы.
* Room-индекс фотографий ещё не реализован.
* Карта и кластеризация ещё не реализованы.
* Галерея и полноэкранный просмотр ещё не реализованы.
* WorkManager-сканирование ещё не реализовано.
