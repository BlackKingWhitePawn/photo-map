# Traverse v0.11.0

Date: 2026-07-18

## Added

* New `home` start screen with a vertically scrolling personal geography dashboard.
* Compact MapLibre trip heatmap preview using existing ready trip heat cells.
* Featured trips carousel with cover thumbnails, date ranges, and photo counts.
* Popular places cards built from already indexed local photos with coordinates.
* All-places and place-details routes.
* Place detail screen with photo mini-galleries grouped by day.
* "On this day in the past" section with a map preview and thumbnails.
* Reusable `MiniPhotoGallery` and `MiniPhotoThumbnail` components.

## Changed

* App navigation now starts at the home screen.
* Permission success navigation now returns to the home screen instead of opening the full map directly.
* Android version metadata is now `versionName=0.11.0`, `versionCode=17`.
