# Ветка `fix-09-map-cluster-controls`

## Release Target

```text
v0.6.0
```

## Цель

Исправить управление картой и кластеризацией после патча: убрать лишнюю подпись, заменить иконку настроек, сделать плотность кластеров применяемой при изменении slider и синхронизировать подпись кластера с открываемым списком фотографий.

## Границы работ

В ветку входит:

* стандартная gear-иконка для настроек;
* удаление подписи `Карта` с верхней панели;
* применение плотности кластеров во время изменения slider;
* отмена устаревших пересчетов кластеров при быстром изменении плотности;
* загрузка реальных `photoIds` для видимых кластеров, чтобы подпись и нижняя галерея показывали один и тот же состав;
* сохранение отката Java compatibility на 1.8, чтобы AGP не запускал `androidJdkImage` через `jlink`.

В ветку не входит:

* запуск сборки Codex-ом;
* запуск тестов, lint, tsc, eslint или npx;
* изменение или удаление оригинальных фотографий;
* отправка фотографий, координат или EXIF наружу.

## Проверки

Сборку выполняет пользователь вручную. Codex не запускает build/compile/test после патча.
