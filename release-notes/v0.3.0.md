# Photo Map v0.3.0

Дата: 2026-07-15

Релиз первого экрана карты: приложение показывает фотографии с GPS-координатами на MapLibre-карте без Google Maps API key, Google Play Services, регистрации и billing.

## Добавлено

* Первый экран карты на MapLibre Android SDK.
* Дефолтный OpenFreeMap Liberty style:

```text
https://tiles.openfreemap.org/styles/liberty
```

* Настройка `MAP_STYLE_URL` через `local.properties`.
* Read-only чтение координат из MediaStore и EXIF через `ContentResolver`.
* Валидация координат: диапазоны широты/долготы и отклонение точки `0,0`.
* Отображение фотографий с валидными координатами на карте.
* Базовая группировка близких точек при низком масштабе.
* Первичное центрирование карты по bounding box всех фотографий с координатами.
* Жесты скролла и зума карты.

## Изменено

* Требование Google Maps заменено на MapLibre/OpenFreeMap.
* Слои границ стран скрываются после загрузки стиля карты.
* Экран доступа показывает количество фотографий с координатами и кнопку `Открыть карту`.

## Безопасность

* Приложение продолжает работать только с локальными фотографиями через read-only доступ.
* Исходные фотографии не удаляются, не изменяются, не перемещаются и не перезаписываются.
* Debug APK проверен: `ACCESS_FINE_LOCATION`, `ACCESS_COARSE_LOCATION` и `ACCESS_WIFI_STATE` не запрашиваются.
* Фотографии, координаты, EXIF и идентификаторы файлов не отправляются на сервер приложения.

## APK

Ожидаемое имя release asset:

```text

```

Signed release APK нужно собрать после merge/проверки этой ветки, когда `versionName=0.3.0` и `versionCode=3` уже находятся в коде.
