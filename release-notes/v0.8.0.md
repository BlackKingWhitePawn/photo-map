# Traverse v0.8.0

Date: 2026-07-17

## Added

* Date range filtering on the map.
* A second map FAB opens a two-sided date range slider.
* The date FAB shows the selected range as `dd.MM.yy - dd.MM.yy`.
* The date slider shows the active dragged range as `dd MMMM yyyy`.

## Changed

* Map cluster rebuilding is now a cancellable asynchronous operation for date filtering, density changes, and indexed photo updates.
* Visible viewport cluster loading now cancels stale map movement queries before they can overwrite newer map results.
* Screen-level marker compaction now runs off the main thread and cooperatively cancels during rapid map movement.
* Cluster density slider changes are applied on release, matching the date range slider behavior.

## Fixed

* No separate bug fixes.
