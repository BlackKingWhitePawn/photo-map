# Traverse v1.0.0

Date: 2026-07-18

Traverse v1.0.0 is the first stable release of the local-first Android gallery atlas. It turns an on-device gallery into a private travel atlas: photos stay on the phone, location metadata is indexed locally, and the main experience focuses on places, trips, time, and density rather than a flat camera roll.

Traverse v1.0.0 - первый стабильный релиз локальной Android-карты фотографий. Приложение превращает галерею на устройстве в личный атлас поездок: фотографии остаются на телефоне, координаты индексируются локально, а главный экран показывает места, поездки, даты и плотность снимков вместо обычной ленты.

## Added

**Русский**

* Локальная карта галереи на MapLibre: снимки с GPS отображаются на карте без Google Maps API key.
* Индексация геоданных из MediaStore/EXIF с повторным использованием уже обработанных координат.
* Кластеризация близких фотографий, одиночные маркеры с миниатюрами и тепловые слои плотности.
* Главный экран Traverse с географией галереи, теплокартой поездок, карточками поездок, популярными местами и блоком "В этот день".
* Карта поездок на основе локально найденных маршрутов и остановок.
* Экраны всех мест и деталей места с мини-галереями, сгруппированными по дням.
* Экспорт карточки поездки в PNG для отправки через системный share sheet.
* Отдельная иконка шестеренки для настроек на основных экранах.

**English**

* Local MapLibre gallery map: geotagged photos are shown on a map without a Google Maps API key.
* On-device MediaStore/EXIF location indexing with reuse of previously processed coordinates.
* Nearby-photo clustering, thumbnail markers for single photos, and density heatmap layers.
* Traverse home screen with gallery geography, trip heatmap, trip cards, popular places, and an "On this day" block.
* Trip map built from locally detected routes and stops.
* All-places and place-detail screens with day-grouped mini galleries.
* Trip summary PNG export through the system share sheet.
* Dedicated gear icon for settings entry points.

## Changed

**Русский**

* Видимое название приложения обновлено на Traverse.
* Верхний виджет главного экрана теперь показывает настоящую теплокарту поездок, а не декоративный фон.
* На миникарте главного экрана скрыты границы стран и подписи стран, чтобы виджет выглядел как часть продукта, а не как политическая карта.
* Экспорт поездки использует первую фотографию поездки как фон и убирает метрику "уверенность".
* Нижняя панель карты стала ровнее: дата и переключатель режимов имеют одинаковую высоту, а режимы "теплокарта" и "фото" оформлены как сдвоенная кнопка.
* Настройки стали проще: параметры алгоритма, debug-блок и экспорт диагностических логов убраны из пользовательского интерфейса.
* Android metadata обновлена до `versionName=1.0.0`, `versionCode=19`.

**English**

* The visible app name is now Traverse.
* The top home widget now renders the real trip heatmap instead of a decorative background.
* Country borders and country labels are hidden on the compact home map so the widget reads as product UI rather than a political map.
* Trip export uses the first photo from the trip as the background and removes the confidence metric.
* Bottom map controls are cleaner: date and mode controls share the same height, and heatmap/photo modes are presented as a two-part segmented button.
* Settings are simpler: algorithm tuning, the debug block, and diagnostic log export are removed from the user-facing screen.
* Android metadata is now `versionName=1.0.0`, `versionCode=19`.

## Fixed

**Русский**

* Заменены оставшиеся иконки в стиле гаечного ключа на шестеренку.
* Исправлен отсутствующий импорт Compose `Color` в контролах карты.
* Подготовленный APK проверен по версии, label, permissions, zipalign и подписи.

**English**

* Replaced the remaining wrench-style settings icons with a gear icon.
* Fixed the missing Compose `Color` import in map controls.
* The prepared APK was checked for version metadata, label, permissions, zip alignment, and signing status.
