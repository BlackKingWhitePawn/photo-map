# Photo Map v0.7.0

Date: 2026-07-16

## Added

* Explicit permission request handling when required photo or media-location access is missing.
* Mini gallery for large map clusters with horizontal scrolling.
* Animated cluster-density FAB that shows the current density percent and expands into a full-width slider.

## Changed

* Removed the map title and back button from the map screen.
* Moved map settings and centering into compact icon-only top-bar controls.
* Colored the status-bar safe area with the top-bar surface and padded controls around status bars and display cutouts.
* Replaced custom zoom/settings assets with standard Android drawable icons.
* Removed the scan action from settings.
* Cluster density now affects runtime visible clustering and the calculated distance limit.
* The map debug panel defaults to hidden for new installs.

## Fixed

* Tapping a cluster refreshes an already-open mini gallery with that cluster's photos.
* Cluster labels are based on real visible photo IDs when available.
* Mini gallery controls are simplified: close is an icon button, zoom and load-more actions are removed.
* Java compile compatibility remains at 1.8 so this project setup does not require the AGP `androidJdkImage`/`jlink` transform.
