# Photo Map v0.7.2

Date: 2026-07-17

## Added

* No new user-facing features.

## Changed

* Map overlay markers keep their cluster composition stable while the camera is moving.
* Nearby dense marker groups compact into larger screen clusters after the normal clustering pass.
* First-time gallery permission grants now start the EXIF GPS scan immediately.
* EXIF scans now publish saved batches to the map while scanning is still running.

## Fixed

* Tapping a cluster whose photos share the same geotag now opens the mini gallery immediately instead of repeatedly zooming into the same point.
* Overlay markers now reproject during pan, pinch, pitch, and zoom gestures so they remain attached to the map instead of the screen.
* Full cluster recalculation now waits until camera idle, reducing visible marker jumping while the user is moving the map.
* Single-photo map markers now render as rounded square thumbnails instead of circular thumbnails.
* Cancelled EXIF scans now resume from the last saved scan mark when started again, if the MediaStore order is unchanged.
